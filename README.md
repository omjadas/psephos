# electionSimulator

[![Actions Status](https://github.com/omjadas/electionSimulator/workflows/Node%20CI/badge.svg)](https://github.com/omjadas/electionSimulator/actions)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```
